# tfmoduleschema

A Go library and CLI that returns **variables, outputs, required providers,
versions, and submodules** for any Terraform module published on the OpenTofu
or HashiCorp Terraform registries.

Sibling project to [`tfpluginschema`](https://github.com/matt-FFFFFF/tfpluginschema),
but for modules instead of providers.

Status: **under construction** — see [`PLAN.md`](PLAN.md) and [`STATE.md`](STATE.md).
